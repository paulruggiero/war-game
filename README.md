# War Game

A roguelike deckbuilding card game inspired by War, Balatro, and extraction-style risk systems.

## Current MVP
- One loss ends the run
- War risks your own cards
- Winning War captures loot
- Shop allows adding and destroying cards
- War slots control how many cards are committed

## Files
- `war_extract.js` - terminal-based prototype of the game loop
